## Goal
Ensure all Divi modules follow the project's atomic CSS guidelines.

## Instructions
1. Check the /wp-content/themes/ folder for the active child theme.
2. If generating a new module, use the provided PHP template in /resources/MODULE_TEMPLATE.php.
3. Validate that no XML-RPC calls are introduced (Security Rule).
